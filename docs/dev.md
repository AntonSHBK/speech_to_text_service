### **üîπ –ö–∞–∫ –∑–∞–≥—Ä—É–∂–∞—Ç—å –∞—É–¥–∏–æ –≤ Whisper?**
Whisper **–Ω–µ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç** —Ñ–∞–π–ª—ã –Ω–∞–ø—Ä—è–º—É—é, –Ω–æ **Hugging Face Pipeline –∏ torchaudio** –ø–æ–∑–≤–æ–ª—è—é—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —Ñ–æ—Ä–º–∞—Ç–æ–≤.

#### **1. –ü—Ä—è–º–∞—è –∑–∞–≥—Ä—É–∑–∫–∞ –≤ `torchaudio` (WAV, FLAC)**


#### **2. –ö–æ–Ω–≤–µ—Ä—Ç–∞—Ü–∏—è —á–µ—Ä–µ–∑ FFmpeg (MP3, M4A, AAC)**
–ï—Å–ª–∏ —É –≤–∞—Å **MP3, M4A, AAC**, —Ç–æ –ª—É—á—à–µ –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å –∏—Ö –≤ WAV –ø–µ—Ä–µ–¥ –ø–æ–¥–∞—á–µ–π –≤ Whisper:
```python
import torchaudio
import torchaudio.transforms as transforms

# –ö–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ–º –≤ 16kHz WAV

```

#### **3. –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ `ffmpeg` –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –ª—é–±—ã–º–∏ —Ñ–æ—Ä–º–∞—Ç–∞–º–∏**
Whisper —Ä–∞–±–æ—Ç–∞–µ—Ç —Å FFmpeg, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–æ–π –æ–±—Ä–∞–±–æ—Ç–∫–∏ –∞—É–¥–∏–æ—Ñ–∞–π–ª–æ–≤:
```bash

```
**–ß—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–æ–º–∞–Ω–¥–∞?**
- `-i input.mp3` ‚Äî –≤—Ö–æ–¥–Ω–æ–π —Ñ–∞–π–ª
- `-ar 16000` ‚Äî —á–∞—Å—Ç–æ—Ç–∞ –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏ 16 kHz (—Ç—Ä–µ–±—É–µ—Ç—Å—è –¥–ª—è Whisper)
- `-ac 1` ‚Äî –º–æ–Ω–æ (–æ–¥–∏–Ω –∫–∞–Ω–∞–ª)
